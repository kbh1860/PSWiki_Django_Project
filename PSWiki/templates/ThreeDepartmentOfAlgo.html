삼분탐색 알고리즘

- 삼분 탐색은 볼록함수에서 극값 혹은 최대/최솟값을 찾을 때 사용할 수 있는 테크닉으로, 아래로 볼록한 함수의 최솟값, 위로 볼록한 함수의 최댓값을 구할 때 사용합니다




PseudoCode : 

                

                    

                        
                                                                                                    long long ternary_Search(int st, int en) {                                                                                                   

  int l = st, r = en;
  while(r-l >= 3)
  {
      int range = (r - 1) / 3;
      int mid1 = l + range;
      int mid2 = r - range;
      if(arr[mid1] < arr[mid2])
      r = mid2;

      else
      l = mid1;
  }

  long long ret = arr[l];

  for (int i = l + 1; i < r; i++)
     ret = min(ret, arr[i])
  return ret;
